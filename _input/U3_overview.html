Unit 3:
Creating Hyperlinks & Images in HTML

Overview

In Unit 3, we will learn about hyperlinks and how they are used to link to other documents or pages. We will study the anatomy of a hyperlink which includes an anchor element, href attribute, and URL. Then, we will look at the title attribute, which is used to add supporting information to a hyperlink or image.

We will study empty elements such as the image element which is used to add an image into a page. Then, we will look at the alternative text attribute which is textual description used when an image cannot be seen or displayed. Last, we will learn how to add a caption to an image using the figure and figure-caption element.

Objectives

Upon completion of this learning unit you will be able to:

  • Use an anchor element, href attribute, and URL to create a hyperlink.
  • Use an image element and source attribute to embed an image into a document.
  • Use the alternative text attribute to add a textual description to an image.
  • Use the title attribute to add supporting information to a hyperlink or image.
  • Use a figure element and a figure-caption element to add a caption to an image.
